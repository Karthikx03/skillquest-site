# 🎓 SkillQuest
> Learn the Skills School Never Taught You

SkillQuest is a free, gamified learning platform. Earn points, beat quizzes, and build real-world knowledge.

🌐 **Live:** https://karthikx03.github.io/skillquest-site

## Features
- 24 free courses across 6 subjects
- Points, leaderboard & certificates
- 100% free — no credit card ever

## Subjects
Financial Literacy · AI Fundamentals · Cybersecurity · Digital Skills · Career Readiness · Entrepreneurship

## Run Locally
```bash
git clone https://github.com/Karthikx03/skillquest-site.git
cd skillquest-site && npm install && npm start
```
